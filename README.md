# Dataset-augmentation-ComfyUI-workflows
Workflows de ComfyUI para realizar data augmentation sintético mediante modelos de difusión, enfocados en la reidentificación a largo plazo de elementos urbanos.

Este repositorio contiene los *workflows* y la documentación de despliegue de **ComfyUI** utilizados para generar datos sintéticos de elementos urbanos. El objetivo de estas herramientas es realizar *data augmentation* de alta calidad para entrenar modelos de reidentificación a largo plazo.

##Contenido del Repositorio

* **`workflows/`**: Archivos `.json` listos para cargar en ComfyUI, diseñados específicamente para la generación automatizada y aleatoria de identidades y variaciones sintéticas de elementos urbanos.
* **`requirements.txt`**: Listado exacto de las librerías base de Python y las dependencias de los nodos personalizados (Custom Nodes) necesarios para replicar el entorno de generación.
